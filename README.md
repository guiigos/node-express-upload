## Usage
### Environment
Installation
```bash
$ npm install npm -g
```

### Dependencies
Install dependencies and modules
```bash
$ npm install
```

### Run
Compile project
```bash
$ npm start
$ npm run nodemon
```

## Routes
- **POST** */multer* [multer](https://github.com/expressjs/multer)
- **POST** */busboy* [busboy](https://github.com/mscdex/busboy) uploads image to [transfer.sh](https://transfer.sh/)
