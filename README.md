## Utilização
### Ambiente
Instalação
```bash
$ npm install npm -g
```

### Dependências
Instalar dependências e módulos
```bash
$ npm install
```

### Executar
Compilar projeto
```bash
$ npm start
$ npm run nodemon
```

## Rotas
- **POST** */multer* [multer](https://github.com/expressjs/multer)
- **POST** */busboy* [busboy](https://github.com/mscdex/busboy) realiza o upload da imagem para o [transfer.sh](https://transfer.sh/)
